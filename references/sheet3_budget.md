# 3枚目（研究経費の内訳）ガイド

## 大前提

- **直接経費の上限：1課題あたり 500 万円 以下**
- **間接経費：直接経費の30%**（機関に配分、調書には計上しない）
- **人件費は本事業では対象外**
- **再委託費・共同実施費も対象外**
- **消費税相当額は補助事業のため対象外**
- **xlsx 様式の金額欄は「千円」単位で固定**されているため、書き込む時のみ `万円 × 10 = 千円` に換算。人が話す／メモする時は **万円単位** で統一する（500万円=5,000千円のような書き方は使わない）。
- 千円未満は切り捨て
- 費目間流用は**文科省の承認なしで直接経費総額の50%以内**

## 費目構成（府省共通経費取扱区分表）

### 直接経費

| 大項目 | 中項目 | 3枚目の明細欄 |
|---|---|---|
| 物品費 | 設備備品費 | A5:E8（品名・設置機関・数量・単価・金額） |
| 物品費 | 消耗品費 | F5:G8 |
| 人件費・謝金 | 人件費 | **対象外** |
| 人件費・謝金 | 謝金 | A14:B16 |
| 旅費 | 国内旅費 | C14:D16 |
| 旅費 | 外国旅費 | E14:F16 |
| その他 | 外注費／印刷製本費／会議費／通信運搬費／光熱水料／その他（諸経費） | A22:B26 |

### 計上ルール（SPReAD 固有）

- **設備備品費**：GPU 搭載サーバ、ワークステーション、研究用ストレージ装置等で**備品として管理すべきもの**
- **消耗品費**：安価な周辺機器、記録媒体、交換部材等で**消耗品として管理すべきもの**
- **AWS 等民間クラウド利用料（物品購入を伴わない）→ 4枚目「計算資源費用」へ、3枚目では「その他」の「事項」に計上**
- **API 利用料（従量課金 LLM 等） → 4枚目「API費用」へ、3枚目では「その他」に計上**
- 「その他（諸経費）」として、次世代理工系人材育成促進（出前授業、SNSコンテンツ制作 等）の経費も可

## 必要性記載が求められるケース

以下のいずれかに該当する場合、3枚目の該当「必要性」欄に理由を記載：

1. 設備備品費が総額の90%超
2. 旅費が総額の90%超
3. 謝金が総額の90%超
4. 消耗品費・その他費目で**特に大きな割合**を占める経費がある

SPReAD では AWS 利用が「その他」に集中して大半を占めるケースが多い → **A28（その他の必要性）には必ず理由を記載**。

## ヒアリングの進め方

1. 総額目処（500 万円以下、典型 300〜500 万円）
2. 設備備品費を使うか（オンプレGPU購入など）Yes/No
3. 消耗品費（書籍、ソフトウェア、周辺機器 等）
4. 謝金（被験者謝礼、外部講師 等、本事業は研究期間が短いので想定少なめ）
5. 旅費（学会参加、現地調査 等）
6. その他 — ここに AWS 費用総額（＝4枚目の合計）と API 費用を配分

AWS 中心案の典型配分例（500万円 ＝ 上限いっぱい）：
- 設備備品費：0 万円
- 消耗品費：20 万円（書籍・周辺機器）
- 謝金：0 万円
- 国内旅費：30 万円（学会参加1回）
- その他：450 万円（うち AWS 計算資源 400 万円、Bedrock API 50 万円）
- 合計：500 万円

## 書き込み時の注意

- `E9`（設備備品費総計）、`G9`（消耗品費総計）、`B17`（謝金総計）、`D17`（国内旅費総計）、`F17`（外国旅費総計）、`B27`（その他総計）— これらは **数式で SUM する**か、手動合計を入れるか選べるが、**1枚目の総計欄（C30）がこれらを参照している**ので、入力した明細と総計が一致している必要がある。

推奨：各明細行の金額セルに数値を入れ、総計セルは `=SUM(...)` で合計する。

- 3枚目は**行追加が許可されている**ので、品目数が多い場合は行を insert する（ただし総計行より上に追加し、総計の数式範囲を拡張する）。

## 単位換算の早見表

ユーザーとのやり取りは **万円** で統一、xlsx 様式への書き込みは **千円** で行う。

| 万円 | 千円（xlsx 書込値） |
|---:|---:|
| 10 万円 | 100 |
| 50 万円 | 500 |
| 100 万円 | 1,000 |
| 300 万円 | 3,000 |
| 500 万円（上限） | 5,000 |

換算式：`千円 = 万円 × 10`
