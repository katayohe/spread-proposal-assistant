# 2nd_form1_proposal.xlsx（第2回_様式1_研究計画調書.xlsx）構造リファレンス

第2回 SPReAD 公募の様式1テンプレート（スキル同梱の `call_materials/application_forms/2nd_form1_proposal.xlsx` ＝日本語版／`2nd_form1_proposal_en.xlsx` ＝英語版）の全シート構造。応募者が編集するのは**4枚**（1枚目〜4枚目）。第1回で存在した「5枚目（AIインタビュー実施完了報告書）」は第2回では廃止された。

入力セルの位置（座標）は日本語版・英語版で共通。

## 保護された数式（絶対に上書きしない）

### 1枚目

- `D48` = `=SUM(E48:L48)` — 研究経費の総計（千円）
- `E48` = `=研究計画調書_3枚目!J31` — 設備備品費合計
- `G48` = `=研究計画調書_3枚目!N31` — 消耗品費合計
- `I48` = `=研究計画調書_3枚目!E60` — 謝金合計
- `K48` = `=SUM(研究計画調書_3枚目!$J$60,研究計画調書_3枚目!$N$60)` — 国内＋外国旅費合計
- `L48` = `=研究計画調書_3枚目!E89` — その他費用合計
- `N35` = `=LEN(C35)` — 研究課題名 文字数カウント
- `N42` = `=LEN(C42)` — 現在のAI活用方法 文字数カウント

### 2枚目

- `E8` = `=LEN($D8)`（英語版は単語数カウント式）— 研究目的
- `E9` — 研究方法 文字数/語数
- `E10` — AI利活用の妥当性
- `E11` — 達成目標
- `E12` — ノウハウ抽出
- `E13` — 成果の公開方針(任意)
- `E14:E18` — 業績欄の文字数チェック式（merged）

### 3枚目

- 設備備品費 行ごと: `J{row}` = `=G{row}*I{row}` (rows 11..30)
- `J31` = `=SUM(J11:J30)` — 設備備品費 小計
- `N31` = `=SUM(N11:N30)` — 消耗品費 小計
- `E60` = `=SUM(E40:E59)` — 謝金 小計
- `J60` = `=SUM(J40:J59)` — 国内旅費 小計
- `N60` = `=SUM(N40:N59)` — 外国旅費 小計
- `E89` = `=SUM(E69:E88)` — その他費用 小計

| テーブル | 行範囲 | 列 |
|---|---|---|
| 設備備品費 | 11〜30 | D=事項, E=設置機関(merged E:F), G=単価千円(merged G:H), I=数量, J=金額千円(数式 G*I) |
| 消耗品費 | 11〜30 | M=事項, N=金額千円 |
| 謝金 | 40〜59 | D=事項, E=金額千円 |

必要性記入欄（merged C..N の左上）:
- `C34` 設備備品費・消耗品費の必要性
- `C63` 謝金・旅費の必要性
- `C92` その他費用の必要性

各費目の小計セル（テンプレ式）と1枚目総計セルは編集禁止。

### 4枚目（千円単位、千円未満切り捨て）

API費用テーブル: 行9〜18（最大10行）
- C: # （連番、テンプレ済み）
- D: 処理対象
- E: 金額（千円）
- F: 算定根拠（merged F:G）

計算資源費用テーブル: 行22〜31（最大10行）
- C: # （連番）
- D: GPU種類
- E: 選定理由
- F: 金額（千円）
- G: 算定根拠

設備備品費として計算資源を計上する場合（GPUカード単独購入等）も4枚目に詳細を書く（注意事項に明記）。

## 編集ルール

- 色付きセル（薄オレンジ）＝未入力。ただし必須項目だけでなく任意項目もあるので、全てを埋める必要はない。
- レイアウト・書式の変更禁止。**3枚目も第2回からは行追加禁止**（20行固定枠）。
- PDF 化せず .xlsx のまま提出。
- ファイル名:
  - 日本語版: `第2回_様式1_研究計画調書_{e-Rad所属機関コード}_{ローマ字氏名}.xlsx`
  - 英語版（任意）: `2nd_Form1_Research Plan_{e-Rad Institution code}_{Name}.xlsx`
- 直接経費は10万円以上500万円以下。
- 設備備品費・旅費・謝金のいずれかが90%超の場合は3枚目の必要性欄に明記必須。
- 色を付した図や文字もそのまま審査に付されるので、適切に使う。
