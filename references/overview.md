# form1_proposal.xlsx（様式1＿研究計画調書.xlsx）構造リファレンス

スキル同梱の `call_materials/application_forms/form1_proposal.xlsx`（元ファイル名：`様式1＿研究計画調書.xlsx`）の全シート構造。応募者が編集するのは5枚（1枚目〜5枚目）。

## シート一覧

| シート名 | 役割 | 応募者が編集 |
|---|---|---|
| はじめにご確認ください）記入にあたっての留意事項 | 記入要領 | × |
| 参考）府省共通経費取扱区分表 | 費目区分表 | × |
| 研究計画調書_1枚目 | 基本情報 | ○ |
| 研究計画調書_2枚目 | 研究目的・方法等 | ○ |
| 研究計画調書_3枚目 | 研究経費の内訳 | ○ |
| 研究計画調書_4枚目 | API・計算資源の算定根拠 | ○（必要な場合のみ） |
| 研究計画調書_5枚目 | AIインタビュー完了証跡貼付 | ○（スクショ貼付） |
| リスト | ドロップダウン用マスタ | × |

## 保護された数式（絶対に上書きしない）

### 1枚目

- `C30` = `=SUM(D30:L30)` — 研究経費の総計
- `D30` = `=研究計画調書_3枚目!E9` — 設備備品費合計をリンク
- `F30` = `=研究計画調書_3枚目!G9` — 消耗品費合計
- `H30` = `=研究計画調書_3枚目!B17` — 謝金合計
- `J30` = `=SUM(研究計画調書_3枚目!D17,研究計画調書_3枚目!F17)` — 国内＋外国旅費
- `L30` = `=研究計画調書_3枚目!B27` — その他合計
- `L22` = `=LEN(B22)` — 研究課題名（日本語）文字数
- `M22` = 英語ワード数カウント式
- `L26`, `M26` — 現在のAI活用方法 文字数

### 2枚目

- `B3` = `=LEN(A3)` — 研究目的（日本語）文字数
- `C3` — 研究目的（英語）ワード数
- `B5, C5` — 研究方法の文字数
- `B7, C7` — AI利活用の妥当性の文字数
- `B9, C9` — 達成目標の文字数
- `B11, C11` — ノウハウ抽出の文字数
- `B15, C15` — 研究業績（5件以内チェック式）

### 4枚目（千円単位、千円未満切り捨て）

- API費用テーブル：行5〜14（最大10行）
  - B列：処理対象
  - C列：金額（千円）
  - D列：算定根拠
- 計算資源費用テーブル：行18〜27（最大10行）
  - B列：GPU種類
  - C列：選定理由
  - D列：金額（千円）
  - E列：算定根拠

### 5枚目

- A3：メールアドレス確認チェック（True/False）
- A4 付近：AIインタビュー完了メールのスクリーンショット画像を貼付

## 編集ルール

- 色付きセル（薄オレンジ）＝未入力。ただし必須項目だけでなく任意項目もあるので、全てを埋める必要はない。
- レイアウト・書式の変更禁止。3枚目のみ行追加可。
- PDF 化せず .xlsx のまま提出。
- ファイル名は `様式１＿研究計画調書＿{e-Rad所属機関コード}＿{ローマ字氏名}.xlsx`。
